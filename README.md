# flask-starters
 starter file for flask application

## steps for starting the file

1. create virtual environment

```
py -m venv venv
```

or

```
python -m venv venv
```

2. start your venv

```
.\venv\Scripts\activate
```

3. install the requirements 

```
pip install -r requirements.txt
```

4. start the project

```
py app.py
```

or

```
python app.py
```
